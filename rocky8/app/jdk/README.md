# OPEN JDK
OPEN JDK

## 1. Install

### 1.1 Install JDK 8

    dnf -y install java-1.8.0-openjdk java-1.8.0-openjdk-devel
    
    java -version
            
### 1.2 Install JDK 11

    dnf -y install java-11-openjdk java-11-openjdk-devel
    
    java -version

### 1.3 Alternative

    alternatives --config java
    
    alternatives --config javac
